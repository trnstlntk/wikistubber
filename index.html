Welcome to WikiStubber!

This tool helps you create Wikipedia articles based on existing Wikidata entries. We'll guide you through selecting a subject, choosing a language, and generating a Wikipedia draft that you can further develop.

For this initial version, we're focusing on NADD creators - a collection documented in Wikidata with property P5008 and value Q133504755.

How it works:

  1. Browse and select a Wikidata item about a NADD creator
  2. Choose which language Wikipedia you want to create the article for
  3. Generate a Wikipedia stub based on Wikidata information
  4. Save the stub to your Wikipedia user space to continue working on it
